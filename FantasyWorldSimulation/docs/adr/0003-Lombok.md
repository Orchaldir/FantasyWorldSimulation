# ADR 0003 - Use Lombok to avoid repetitive code

**Status:** accepted

**Date:** 2018-09-30

## Decision Outcome

* Avoid writing
    * getters & setters
    * constructors
    * equals()
    * hashCode()
    * builders
    * loggers with **@Slf4j**
* Requires ide support.
