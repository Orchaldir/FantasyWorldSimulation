# ADR 0005 - Use SLF4J for logging

**Status:** accepted

**Date:** 2018-10-08

## Context and Problem Statement

How to log?

## Considered Options

* System.out()
* LOG4J
* SLF4J

## Decision Outcome

**Chosen option:** SLF4J

* **{}** is less error prone.
* Better integration with libraries.
* Use @Slf4j from [Lombok](0003-Lombok.md).
